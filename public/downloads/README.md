# 7-Day Clarity Sprint Guide

This directory should contain the downloadable PDF lead magnet: `7-day-clarity-sprint.pdf`

## Creating Your Lead Magnet

To create your 7-Day Clarity Sprint PDF, you can:

1. **Use Canva** (Recommended for non-designers):
   - Sign up at https://www.canva.com
   - Search for "Workbook" or "Guide" templates
   - Customize with your brand colors (#0B3D3C, #E8B44D, #D9462E)
   - Export as PDF

2. **Use Google Docs/Slides**:
   - Create a structured 7-day program
   - Add exercises, prompts, and worksheets
   - Export as PDF

3. **Hire a Designer** (Professional option):
   - Fiverr, Upwork, or 99designs
   - Budget: $50-300 depending on complexity

## Content Suggestions for 7-Day Clarity Sprint

**Day 1: Values Audit**
- Exercise to identify top 5 core values
- Reflection prompts about alignment

**Day 2: Strengths Assessment**  
- Worksheet to identify unique strengths
- Examples of how to leverage them

**Day 3: Vision Crafting**
- 3-5 year career vision exercise
- Visualization prompts

**Day 4: Gap Analysis**
- Current vs. desired state mapping
- Obstacle identification

**Day 5: Quick Win Strategy**
- Identify 3 immediate action items
- 30-day goal setting

**Day 6: Confidence Building**
- Past success inventory
- Confidence-building affirmations

**Day 7: Action Plan**
- Next 90-day roadmap
- Accountability framework
